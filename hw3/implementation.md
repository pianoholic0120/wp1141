# NTU Course Registration Service - shadcn/ui Component Mapping

## Stage 1: Browse & Explore

### Main Layout Components
- **Navigation Menu** - Main navigation header with stage indicators
- **Sidebar** - Filter panel on the left side
- **Scroll Area** - Course catalog list/grid container
- **Separator** - Visual section dividers

### Course Catalog Components
- **Card** - Course information cards (grid view)
- **Table** - Detailed course list view
- **Badge** - Course tags (department, level, credits)
- **Avatar** - Instructor profile pictures
- **Aspect Ratio** - Course image placeholders

### Filtering & Search Components
- **Input** - Search bar for course names/codes/instructors
- **Select** - Department, course level, credits dropdowns
- **Checkbox** - Multi-select filters
- **Toggle Group** - View mode switcher (grid/list)
- **Button** - Filter reset, apply actions

### Interaction Components
- **Hover Card** - Course details on hover
- **Tooltip** - Additional info for icons/buttons
- **Pagination** - Course list navigation
- **Skeleton** - Loading states for course cards

### Premium Enhancements
- **Carousel** - Featured courses slider
- **Progress** - Search/filter loading indicators
- **Alert** - System notifications

---

## Stage 2: Planning Container (Staging)

### Calendar & Schedule Components
- **Calendar** - Weekly schedule view
- **Card** - Draggable course blocks
- **Badge** - Conflict warnings, course tags
- **Popover** - Course details on click

### Planning Management Components
- **Tabs** - Multiple schedule scenarios (Plan A, B, C)
- **Accordion** - Collapsible course sections
- **Scroll Area** - Planning list panel
- **Drag and Drop** - Custom draggable course blocks

### Interactive Controls
- **Button** - Add/remove course actions
- **Toggle** - Schedule visibility options
- **Slider** - Time range adjustments
- **Switch** - Auto-conflict detection toggle

### Status & Feedback Components
- **Progress** - Credit counter visualization
- **Badge** - Total credits, warnings
- **Alert** - Conflict notifications
- **Sonner** - Toast notifications for actions

### Premium Enhancements
- **Chart** - Credits distribution visualization
- **Resizable** - Adjustable panel sizes
- **Context Menu** - Right-click course actions
