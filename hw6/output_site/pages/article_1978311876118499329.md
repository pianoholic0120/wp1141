# 四把椅子劇團： 與會員們來場真心（或幽默）的交換吧！

收藏

**【 不止在劇場，讓我們跟會員歡喜相遇吧！】專題 ｜OPENTIX編輯臺 2025年10月**

  

文字 / 郝妮爾

圖片提供 / 四把椅子劇團

  

2018年，成軍10年多的四把椅子劇團（下簡稱四把椅子）開啟的首次長銷演出，在水源劇場連演12場，那是近乎1個月的檔期，也讓團隊進一步思考自己與觀眾的連結，似乎不再僅止於演出時的幾個鐘頭，而期待能夠綿延成一種更深的狀態。因此「讓團隊正是建立起會員制」的想法便應運而生。

  

此後，會員制的招募變成為四把椅子演出現場的「限定申請」條件，通常只在演出現場發生，邀請觀眾在戲前戲後申請入會。然而，入會以後到底能夠「得到」什麼？所謂的「購票優惠」、「演出節目資訊」……對團隊來說都只是基本門檻，他們左思右想，若真的有什麼是會員專屬的「好康」，大概就是讓會員更近一步，走向團員們的日常吧？

  

台上的他們，面對劇場藝術必然肅然莊重，但下了台以後，則盡可能從容幽默。會員制的成立，是給觀眾一把鑰匙，窺見藝術家生活中的另一個面向。

  

![](https://s3.resource.opentix.life/upload/article/img/1760514833542kzd6DoNnD3.jpeg)  

圖 / 2024《太陽》。( 攝影 黃煌智 )

  

## 神秘椅友活動，到底多神秘？

  

劇團經理吳可雲分享，自己本身也是個「追星仔」，以其了解而言，偶像團體與粉絲的連結性很強，會員制度也相對完整。她說：「可是劇場不是偶像，偶像需要不斷給予夠強的吸引力來作為情感寄託，也需要很長時間維繫經營。但是劇團的重心畢竟還是要放在演出上，再加上一般團隊也不會有那麼多的經濟成本去付出。所以，我們還能夠做什麼？」

  

這個問號，是當初才建構會員制度的時候，經常困擾團員的一點。

  

「我們當時只先確定幾件事：首先，一些實質上的優惠內容是最基本的；接著，我們當然是希望能夠先降低入會門檻，因此暫不收取任何入會費用。最後，在當時招收會員的資訊上，我們寫著『不定期會有神秘椅友的專屬活動』。」

  

事實上，所謂的「神秘」起初是帶著很多不確定性。只知道必然得辦，卻不知道該從何下手。然而，不確知是不是這條資訊的關係，在2018那場首次招募會員的長銷作品中，一共累計的4千8百多的售票數，最後竟一口氣招攬了2千人加入會員，吳可雲說：「大概是將近四成的結果，算是蠻好的開始！」同時，所謂「不定期椅友活動」也從那之後正式規劃啟動。

  

起先，團員們有一個明確的原則，那就是提供觀眾另一個面向，讓大家能夠用不同角度看見創作者。

  

「我覺得，四把椅子的團員都有一個非常清楚的特質，就是『幽默感』。所以後來我們也會想要把這件事反映在會員文案上。像是成為會員明明就是免費的事情，我們硬要寫成『第一波加入會員者，享有25年免費的優惠』。」

  

諸如此類像是有點無厘頭的玩笑，就是團員之間的日常，也是四把椅子決定要邀請觀眾共享的一部分！

  

![](https://s3.resource.opentix.life/upload/article/img/1760514954783AKvGp7I1r0.jpeg)

##  
創作者私下真面目大公開？

  

於是，四把椅子在2019年初，舉辦了第一場會員限定的實體活動：「四把！椅子之友第一屆股東聯歡大會」。

  

吳可雲說：「你從活動名稱就可以感受到團隊的風格，有著很胡鬧的特質。我們煞有其事地辦了一場粉絲見面會，做了一段開場影片、和大家一起遊戲，中間也穿插短劇呈現。從頭到尾僅收取清潔費150元。算是讓我們都留下蠻深刻記憶的第一場活動。」

  

事實上，熟悉四把椅子的觀眾或許也熟悉劇團的幽默調性。過去，在劇團演出的「團團圓圓」系列作品中，就能略見端倪。然而會員活動則是近一步地，將這份幽默的距離拉得更近，好似演員也卸下了「演」的成分，能夠袒露更多一點的「真實」與觀眾互動。

  

吳可雲分享：「事實上，在籌備這類活動的現場，對我來說也更接近四把椅子平時的樣貌。雖然我前面說過，劇團不是偶像團體，可是『看見創作者、表演者』與台上的反差，這件事情本身就很有吸引力，感覺與演員之間也會變得更加親近，加深了人與人的連結感。」

  

中途曾因疫情期間暫停幾年的實體活動，在解封以後，四把椅子在2024年之初舉辦了專為椅友打造的「劇本殺」活動。「那場活動，更像是邀請會員來『看演員們』玩劇本殺，你可以選定要跟從哪一個演員的台詞，去猜測兇手是誰。」吳可雲說，諸如此類的活動，當然都不是為了營利，而是回歸最初的期待——在演出活動以外的時間，仍不會間斷與觀眾的連結。

  

![](https://s3.resource.opentix.life/upload/article/img/17605150184824vdHM0n6Wr.jpeg)  

圖 / 2024_劇本殺。( 照片提供  四把椅子劇團 )

  

## 這是一場交換，而非交易

  

那麼，連結的目的是為了什麼呢？藝術家難道會因為觀眾的喜好，而改變自己的創作意圖嗎？

  

仔細梳理下來，四把椅子的會員機制，其實很像一種心意相投的往來交換。

  

沒錯，是交換，而不是交易。

  

作為一個劇團，會員期待的多半不是偶像的應援小物，或者親簽小卡，而是下一場不虛此行的好戲，也或許，是一種知道自己「沒有愛錯」的驗證。而四把椅子截止目前為止所籌辦的各種實體活動，多少加深了一些與會員之間的連結，然而始終沒有停滯下來的，仍是他們正共同等待下一齣作品的意念。

  

「所以……如果問說藝術家會不會因會員的喜好而影響決定？坦白說，真的不會欸。」吳可雲說。

  

比方說，這幾年，四把椅子因為《好事清單》這齣需要大量跟觀眾互動的獨角戲登場，所以曾陸續配合3位不同主演，而籌備了3場「會員限定的試演場」。

  

之所以是會員限定，並不是因為熟悉的觀眾最能讓演員安心，而是因為，這是他們能夠優先提供給會員的、作為劇團所能送出最珍貴的禮物。畢竟，這裡確實有一群人，需要被真心的好戲來充實，而團隊也永遠需要真實的觀眾，給予誠心的回饋。這往來間的交換，讓會員的價值不會只停留在利益的估算。

  

「因此，短時間內我們應該還是會維持這樣的方式進行：在現場演出時，提供加入會員的資訊。不定期舉辦神秘椅友活動，以及——當然啦，現在加入的話，大概還有18年的時間都可以免入會費喔！」吳可雲這麼說著。
