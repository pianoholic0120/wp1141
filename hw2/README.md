# 滑動方塊覆蓋遊戲

一個基於 React TypeScript 的網頁版滑動方塊謎題遊戲，玩家需要控制多個方塊在限定回合內覆蓋地圖的每一格。

## 🎮 遊戲特色

- **多方塊同時控制**：所有方塊會同時朝指定方向滑動
- **策略性玩法**：需要規劃方塊的移動順序和路徑
- **多難度關卡**：從簡單到專家級別的 10 個關卡
- **星級評分系統**：根據使用步數給予 1-3 星評分
- **進度保存**：自動保存遊戲進度和最佳分數
- **響應式設計**：支援桌面和行動裝置

## 🎯 遊戲規則

1. **目標**：讓所有可動方塊覆蓋地圖的每一格（除了障礙物）
2. **移動**：每次輸入方向時，所有方塊同時朝該方向滑動直到撞到障礙物
3. **回合限制**：每個關卡都有回合數限制
4. **評分**：
   - 3★：使用步數 ≤ 最少步數 + 1
   - 2★：使用步數 ≤ 最少步數 + 5
   - 1★：在回合限制內完成

## 🎮 操作方式

### 桌面版
- **方向鍵** 或 **WASD**：移動方塊
- **Z**：撤銷上一步
- **R**：重新開始關卡
- **H**：顯示提示

### 行動裝置
- **觸控按鈕**：點擊方向按鈕移動
- **滑動手勢**：支援四向滑動操作

## 🎯 關卡設計

### 難度分級
- **Easy**：5×5 到 6×6 地圖，1-2 個方塊
- **Normal**：6×6 到 7×7 地圖，2-3 個方塊
- **Hard**：8×8 地圖，2-3 個方塊
- **Expert**：9×9 到 10×10 地圖，3-4 個方塊

### 關卡特色
- 每關都有獨特的障礙物配置
- 逐步增加複雜度
- 平衡的挑戰性

## 🎮 遊戲機制

### 移動系統
- 方塊會朝指定方向滑動直到撞到障礙物
- 移動順序根據方向確定（避免衝突）
- 支援撤銷和重新開始

### 評分系統
- 基於使用步數的星級評分
- 保存最佳成績
- 進度追蹤

## 🎨 視覺設計
